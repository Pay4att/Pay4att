### About Me

- I build production-oriented systems instead of one-off demos.
- My work is mainly around industrial vision, radar data pipelines, and speech inference infrastructure.
- I care about runtime stability, device integration, service delivery, and maintainable deployment paths.

### Current Work

- `SmokeAI`: industrial vision application with desktop management UI, model packaging, licensing, device integration, and runtime inference.
- `Radar`: multi-site radar measurement service for UDP point-cloud ingestion, region analysis, structured outputs, and downstream delivery.
- `xiaozhi_tts_api`: speech serving stack built around IndexTTS and vLLM with Dockerized deployment and OpenAI-compatible APIs.
