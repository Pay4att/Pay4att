## Hello

你好，我是 `Pay4att`。

这个主页主要用来整理我当前在做的几类工作：工业视觉、雷达数据处理、语音服务基础设施，以及一些偏工具化的个人项目。公开仓库保持克制，私有工程则更偏向实际落地和持续迭代。

## Focus

| Direction | Notes |
| --- | --- |
| Industrial Vision | `SmokeAI` 方向，包含桌面管理界面、模型封装、授权机制、设备集成与推理运行链路。 |
| Radar Data Service | `Radar` 方向，处理多站点部署、UDP 点云接入、区域分析、PCD 与 JSON 产出、HTTP 下游推送。 |
| Speech Infrastructure | `xiaozhi_tts_api` 方向，围绕 IndexTTS 与 vLLM 构建语音服务接口与部署能力。 |
| Public Tooling | `HandWritingLatex`、`obsidian-releases` 等公开仓库，偏向工具、实验与生态参考。 |
