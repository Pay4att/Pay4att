# Hello

### About Me

你好，我是 `Pay4att`，主要在做几类工程方向的工作：工业视觉、雷达数据服务、语音推理基础设施，以及一些偏工具化的个人项目。

目前公开仓库不多，但私有项目持续在迭代，重点集中在实际系统落地、桌面应用、模型推理链路、设备接入和服务部署。

我更偏好把项目做成可运行、可维护、可继续扩展的系统，而不是只停留在演示层面。

### Current Focus

- `SmokeAI`：工业视觉应用，覆盖 PyQt 桌面界面、模型封装、授权机制、设备联动与运行时推理。
- `Radar`：多站点雷达测量服务，处理 UDP 点云接入、区域分析、PCD/JSON 输出与 HTTP 下游推送。
- `xiaozhi_tts_api`：基于 IndexTTS 与 vLLM 的语音服务接口，包含 Docker 化部署和 OpenAI 兼容 API。
- `HandWritingLatex`：公开实验仓库，聚焦手写数学、LaTeX 处理与相关工具探索。
