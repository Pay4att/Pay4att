# Pay4att

### Overview

我主要在做工程化落地导向的系统开发，当前工作集中在工业视觉、雷达数据服务、语音推理基础设施，以及围绕这些场景延展出的工具链与桌面应用。

相比只展示概念验证，我更关注系统能否真正运行、维护和持续扩展，包括模型封装、设备接入、服务编排、部署交付，以及面向实际使用场景的稳定性设计。

### Repository Summary

**Public repositories**

- [`HandWritingLatex`](https://github.com/Pay4att/HandWritingLatex): 面向手写数学与 LaTeX 处理的公开实验仓库，包含应用端、插件侧和 OCR 相关探索。
- [`obsidian-releases`](https://github.com/Pay4att/obsidian-releases): 基于上游项目的公开 fork，用于跟踪和处理 Obsidian 插件与主题发布目录相关工作。
- [`Pay4att`](https://github.com/Pay4att/Pay4att): 当前个人主页仓库，用于呈现公开项目、工程方向和持续更新的 GitHub 信号。

**Private repositories**

- `SmokeAI`: 工业视觉应用，覆盖桌面管理界面、模型封装、授权机制、设备联动与运行时推理。
- `Radar`: 多站点雷达测量服务，负责 UDP 点云接入、区域分析、结构化结果输出与下游推送。
- `xiaozhi_tts_api`: 基于 IndexTTS 与 vLLM 的语音服务接口，包含 Docker 化部署与 OpenAI 兼容 API。

### Focus Areas

`Industrial Vision` · `Radar Data Services` · `Speech Infrastructure` · `Desktop Engineering` · `Model Serving` · `Device Integration`
